# おこづかいルーレット

子供がプリントを1枚終えたあとに、ゲームセンター気分でおこづかい抽選を楽しめる Canvas 製ルーレットアプリです。

## 起動方法

外部ライブラリは使用していないため、`index.html` をブラウザで開くだけで動作します。

```text
index.html
```

ローカルサーバーで確認する場合は、プロジェクトのルートで次のように起動できます。

```bash
python3 -m http.server 8000
```

その後、ブラウザで以下を開いてください。

```text
http://localhost:8000
```

## ファイル構成

```text
/
├── index.html
├── style.css
├── script.js
└── README.md
```

## 主な機能

- Canvas によるルーレット描画
- 内部抽選と停止位置の同期
- requestAnimationFrame による 4〜5 秒の減速アニメーション
- Web Audio API によるクリック音、カチカチ音、停止音、当たり音、ジャックポット音
- 100円、500円、1000円の特別演出
- localStorage による統計保存
- iPhone Safari、Android、PC 向けレスポンシブ対応
